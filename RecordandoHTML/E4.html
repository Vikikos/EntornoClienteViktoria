• URL(Uniform Resource Locator) es una dirección única en Internet que identifica y ubica un recurso específico, como una página web, una imagen o un video

  • HTTP(Hypertext Transfer Protocol)es el protocolo estándar de Internet que permite la comunicación entre un navegador web (cliente) y un servidor

  • TCP/IP (Protocolo de Control de Transmisión/Protocolo de Internet) es un conjunto de protocolos estándar de la capa de transporte TCP y de la capa de red IP, en los que se basan las comunicaciones en internet y que cubren los distintos niveles del modelo OSI.

  • Internet es una red de computadoras interconectadas entre sí a nivel mundial con el objetivo de hacer común información de acceso público

  • W3C(World Wide Web) es una comunidad internacional dedicada a desarrollar estándares para la web, como HTML y CSS, para asegurar su crecimiento a largo plazo, interoperabilidad y accesibilidad.

  • HTML(HyperText Markup Language) es el código estándar que define la estructura y el contenido de las páginas web.
